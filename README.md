用友GRP-U8 bx_historyDataCheck.jsp SQL注入漏洞检测脚本

请输入 -u 参数指定 URL 地址：python3 poc.py -u url

usage:

python3 poc.py -u http://xxxx/

python3 poc.py -f file.txt

参数:

  -u URL, --url URL     测试单条url
  
  -f FILE, --file FILE  测试多个url文件
